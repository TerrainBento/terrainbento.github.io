What is Landlab?

terrainbento is a modular landscape evolution modeling package built on top of the Landlab Toolkit.

terrainbento is described in more detail by Barnhart et al. in a Creative Geoscientific Model Development Discussion paper.

Installing the release version of terrainbento with conda

Prebuilt binaries for terrainbento that support multiple Python versions, and the Mac, Linux, and Windows operating systems are distributed through Anaconda Cloud<\a>. Check the hyperlink to identify if we support your Python and OS needs. If we don't, please make a Github Issue<\a>. <\p.

To install terrainbento, you will need a Python Python 2.6+ or 3.5+ distribution. If you are unsure of what you want, we recommend the Anaconda<\a> 3.6 or 3.7 distributions. You will also need the conda package manager<\a>.
